# Instructions

The instructions for running **Programming Question 2** of HW 1

The source code for the programming portion is in the file **`HW1.py`**.



## Dependencies

This script uses `numpy` and `matplotlib`.

Install these dependencies using :

```bash
pip install numpy matplotlib
```
## Execution Instructions
Navigate to the code directory and run 
```
python HW1.py
```

### Notes
* Please note that the code performs approx 11.4 million episodes (specially in 2d). The total runtime may take several minutes. Please be patient.
* The graph for 2a is not displayed on terminal, it comes in a display window. Make sure to save and close it, to view other outputs.
